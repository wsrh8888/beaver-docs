# 后台管理系统概述

海狸IM后台管理系统是基于Vue 3 + Element Plus + TypeScript开发的管理平台，用于管理海狸IM的用户、聊天、群组等功能。

## <img src="/image/manager/dashboard.svg" alt="项目特色" style="width: 32px; height: 32px; vertical-align: middle; margin-right: 8px; display: inline-block;">项目特色

### 核心功能

- <img src="/image/manager/emoji.svg" alt="表情包" style="width: 20px; height: 20px; vertical-align: middle; margin-right: 8px; display: inline-block;"> **表情包管理**: 表情包审核、分类管理、版本控制
- <img src="/image/manager/update.svg" alt="升级" style="width: 20px; height: 20px; vertical-align: middle; margin-right: 8px; display: inline-block;"> **升级管理**: 应用更新发布、版本控制、灰度发布

### 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue.js | 3.5+ | 前端框架 |
| TypeScript | 5.8+ | 类型安全 |
| Element Plus | 2.10+ | UI组件库 |
| Vite | 7.0+ | 构建工具 |
| Pinia | 3.0+ | 状态管理 |
