## Reporting a Vulnerability

如发现 Beaver IM 文档站（beaver-docs）存在安全漏洞，请按以下方式**私下**报告，勿公开披露。

### 🚨 立即行动

1. **不要**在 GitHub 公开 Issue 中讨论漏洞细节
2. **不要**在公开论坛或社交媒体传播漏洞信息
3. **请**通过下方联系方式私下报告

## 🔍 安全说明

文档站为静态站点（VitePress），主要风险包括：

- 文档内容中的恶意链接或脚本示例
- 依赖包（npm）已知漏洞
- 部署配置不当导致的访问控制问题

## 🔄 处理流程

1. **初步响应**：24 小时内确认收到
2. **评估修复**：按严重程度 1–7 个工作日
3. **发布修复**：验证通过后更新文档或依赖
