# 服务端 v1.0.0

**发布日期**: 2025-08-09  
**版本类型**: 正式版本

## <img src="/image/changelog/celebration.svg" alt="版本概述" style="width: 32px; height: 32px; vertical-align: middle; margin-right: 8px; display: inline-block;">版本概述

服务端v1.0.0是海狸IM的首个正式版本，提供了完整的微服务架构和即时通讯功能。

## <img src="/image/changelog/sparkles.svg" alt="主要功能" style="width: 32px; height: 32px; vertical-align: middle; margin-right: 8px; display: inline-block;">主要功能

- **微服务架构**: 基于 go-zero 框架，支持水平扩展
- **实时通信**: WebSocket 连接，支持万级并发
- **文件存储**: 支持多种存储后端
- **用户管理**: 完整的用户注册、登录、权限管理

## <img src="/image/changelog/wrench.svg" alt="技术特性" style="width: 32px; height: 32px; vertical-align: middle; margin-right: 8px; display: inline-block;">技术特性

- **数据库**: MySQL 8.0+ 支持
- **缓存**: Redis 6.0+ 缓存
- **服务发现**: etcd 3.4+ 服务注册
- **反向代理**: Nginx 1.18+ 负载均衡
