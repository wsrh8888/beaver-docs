# 后端开发概述

海狸IM后端基于Go语言和go-zero微服务框架构建，采用微服务架构设计，具有高性能、高可用、易扩展的特点。

## 🎯 技术架构

### 核心技术栈

- **编程语言**: Go 1.23+
- **微服务框架**: go-zero
- **数据库**: MySQL 8.0+
- **缓存**: Redis 6.0+
- **服务发现**: ETCD 3.5+
- **WebSocket**: gorilla/websocket
- **认证**: JWT
- **容器化**: Docker
