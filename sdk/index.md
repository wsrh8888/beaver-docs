# 登录接入概述

第三方接入「海狸账号登录」分两层，按场景选其一即可：

```
┌─────────────────────────────────────────┐
│  上层：Web 授权页（beaver-oauth）开源     │  ← 网站 / H5 跳转授权
│  部署独立授权站，内嵌 JS-SDK 组件         │
└──────────────────┬──────────────────────┘
                   │ 使用
┌──────────────────▼──────────────────────┐
│  下层：JS-SDK（@beaver-im/js-sdk）        │  ← Electron / 自行内嵌
│  npm 安装，提供 <beaver-login> 组件      │
│  不开源                                   │
└─────────────────────────────────────────┘
```

| 场景 | 用哪一层 | 开源 | 文档 |
|------|----------|------|------|
| 第三方网站 / H5 | [Web 授权页](/sdk/oauth/)（beaver-oauth） | 是 | 部署授权站，跳转拿 `code` |
| Electron 桌面应用 | [JS-SDK](/sdk/js-sdk) | 否（npm） | 应用内嵌 `<beaver-login>` |

两者登录成功后都是拿到授权码，再由服务端换 Token → [身份验证](/open/api/oauth/authorize)。
