## 🌟 Project Introduction

Beaver IM is a lightweight, high-performance, and security-focused open-source instant messaging system dedicated to private deployment, providing complete instant messaging solutions for enterprises and developers.

### 🎯 Core Features

- 🆓 **Open Source & Free** - Apache-2.0 license, commercial use allowed
- 🔏 **Private Deployment** - Complete data autonomy and control
- 🆚 **Message Delivery Guarantee** - TCP + ACK mechanism ensures message delivery
- 🔐 **Content Security** - End-to-end encrypted transmission
- 💽 **Permanent Storage** - Messages saved permanently without loss
- 📱 **Multi-platform Sync** - Real-time synchronization across all platforms
- 📟 **Full Platform Support** - iOS, Android, Windows, macOS, Linux, Web

### 🏗️ Technical Architecture

- **Backend**: Go + go-zero microservices framework
- **Mobile**: uni-app + Vue 3 + TypeScript
- **Desktop**: Electron + Vue 3 + TypeScript
- **Admin Panel**: Vue 3 + Element Plus + TypeScript
- **Documentation**: VitePress + Vue 3
